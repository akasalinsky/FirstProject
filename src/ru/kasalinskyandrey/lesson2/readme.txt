
Написать консольное приложение простого калькулятора
Цель: изучить работу со строками и паттернами, поробовать методы, закрепить предыдущий материал

Основные команды:
clear - очистить введенное число
exit - завершить работу приложения

пример работы приложения:

Пользователь вводит операцию с двумя числами
1 + 2
Программа: 3 - ответ хранит в состоянии
Пользователь: / 3
Программа: 1 - ответ хранит в состоянии
Пользователь: * 100
Программа: 100 - ответ хранит в состоянии
Пользователь: clear
Программа: удаляет хранимое в состоянии значение
Пользователь: 10 / 5
Программа: 2 - ответ хранит в состоянии
Пользователь: exit
Программа завершается

Должны быть проверки вводимых строк, извлечение из них чисел и применение к ним мат операций
Должен быть switch, программа должна работать бесконечно пока пользователь не введет exit
Должны использовать методы, а не все тело в main-е
Переменны и методы должны быть названы грамотно, приложение должно лежать в пакете lesson2
